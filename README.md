
# eCome-Store
is an online shopping store, the user can show the products, search for product by writing the name in the search bar or by voice, add to cart, add to wishlist, and make an order and receive an email with order details. (using java, firebase, mvvm architecture pattern). </br>

<span>
<h3>eCom Store futures</h3></br>
1.using MVVM architecture pattern with live data</br>
2.firebase Storage for images and firebase database for the rest of the data</br>
3.using firebase caching capability </br>
4.consist of 10 activity and 7 fragment </br>
5.using the third party library (butter knife, Picasso,  Okhttp)</br>
6.using (sdp, ssp) for views and texts sizes </br>
7.using (taps and view pager for a fragment )</br>
8.using navigation view to view the different pages</br>
9.min SDK is Lollipop</br>
10.using image Compression technique to reduce the size of the image </br>
11.update previous version is allowed if there is a newer version</br>
12.app support multi language</br>

<h3>what user can do</h3></br>

1.Login as User</br>
2.Remember me, forget password</br>
3.Register as User</br>
4.change products view from linear Layout to grid layout (Recycle view, adapter)</br>
5.sort products by price Ascending or descending</br>
6.user can add the product to Wishlist and remove it easily</br>
7.user can add the product to Cart and remove it easily</br>
8.user can view the product in the beautiful view (using the slider for images )</br>
9.Real-time search by (anything in it)</br>
10.voice search </br>
11.user can make an order and specify its quantity</br>
12.order summary page that shows all cost</br>
13.user should specify his current location and can see his location(Location service )</br>
14.user can track his order status</br>
15.user can show pi-chart shows the number of product in each category in his orders (any-chart library, bottom sheet)</br>
16.user can search in the order list </br>
17.user can show its wish list </br>
18.user can logout</br>
<h3>what admin can do</h3></br>
1.login</br>
2.Remember me</br>
3.forget password
4.view its orders and show the customer location</br>
5.can approve the order put it in shipped status and then can put it in delivered status</br>
6.admin can add product delete it and update </br>
7.admin can logout </br>
</span></br>
